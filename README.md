# シンプルパズル

スライド式の数字並べ替えパズルゲーム（15パズル系）。AI機能によるヒント・最適解表示・プレイスタイル分析を搭載。

## 主要機能

- **複数のパズルサイズ**: 4×4（初級）、5×5（中級）、9×9（上級）、カスタムサイズ対応
- **2つの表示モード**: 数字モード、画像モード（画像アップロード可能）
- **シャッフル機能**: 難易度に応じた自動設定、手動調整も可能
- **3つのゲームモード**: タイムアタック、最小手数チャレンジ、フリープレイ
- **AIヒント機能**: Google Gemini API / OpenAI APIによる次の手のヒント
- **AI最適解表示**: A*アルゴリズムによる最短手順の可視化
- **プレイスタイル分析**: AIによる改善点の提案
- **PWA対応**: オフラインでも利用可能

## 技術スタック

- Next.js 14.x / React 18.x
- TypeScript
- Tailwind CSS v3
- Framer Motion (アニメーション)
- Google Gemini API (Primary) / OpenAI API (Fallback)
- IndexedDB (Dexie.js)
- PWA (next-pwa)
- A*アルゴリズム（最適解計算）
